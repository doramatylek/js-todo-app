# js-todo-app
# ToDo приложение

Небольшое веб-приложение для управления списком задач на день.
Позволяет добавлять задачи, отмечать их как выполненные и удалять.
Все данные сохраняются в браузере с использованием LocalStorage.

## Функциональность

* добавление новой задачи
* удаление задачи
* отметка задачи как выполненной
* сохранение списка задач в LocalStorage
* автоматическое восстановление задач после перезагрузки страницы
* отображение состояния пустого списка

## Технологии

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap

## Как работает приложение

1. Пользователь вводит текст задачи.
2. Задача добавляется в список.
3. Список задач хранится в массиве объектов.
4. После каждого изменения данные сохраняются в LocalStorage.
5. При загрузке страницы задачи автоматически восстанавливаются.

## Структура задачи

Каждая задача представляет собой объект:

```javascript
{
  id: number,
  text: string,
  done: boolean
}
```
---

Проект выполнен в учебных целях для практики JavaScript и работы с DOM.
